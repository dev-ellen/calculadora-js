# Calculadora JS

Uma calculadora básica desenvolvida com HTML, CSS e JavaScript. 

## Sobre

Projeto simples de uma calculadora funcional criada para aprender e praticar os fundamentos do desenvolvimento web.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Operações básicas (adição, subtração, multiplicação, divisão)
- Interface intuitiva
- Limpador de valores

## Como usar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Comece a calcular!

## Aprendizados

Este projeto me ajudou a praticar:
- Manipulação do DOM
- Eventos de clique
- Lógica de cálculo
